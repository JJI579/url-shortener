# URL Shortener

As the repo name suggests, it is a URL Shortener.

I didn't make this to be apart of my portfolio, just because I am sick of the current link shorteners which have a problem with you if the link has been copied too many times etc.

With this I can make custom links and everything, whereas others require premium.
